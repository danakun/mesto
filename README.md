# Сервис Место

### Описание
Проект сделан на основе макета из Figma, можно создать свой профиль, поменяв имя и род занятий. 

### Использованные технологии

* Верстка по БЭМ
* Попап-форма на Javascript 
* CSS flex, grid
* Использованы @media для адаптивной верстки
